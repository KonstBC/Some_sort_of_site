И так, создание формы происходит следующим образом: 1) тег form.../form 2) окно для ввода объевляется тегом input, для него не требуется /input 3) объевляете нужный тип данных атрибутом type="" (text, email, password и т.д) 4) и задаёте имя атрибутом name="имя". Так же можно создать в форме тег textarea.../taextarea - поле для ввода текста. Чтобы была и работала кнопка отправить, в типе ввода пишется submit, но вместо атрибута name пишится value="любой текст" (кнопка может переносить пользователя на другую страницу, при условии что в теге form будет написан атрибут action="адрес страницы"). Если вдруг, пользователь не шибко умён, то для него можно написать подсказку, просто вписа в конце тега input атрибут placeholder="email@example.com". Дабавить функцию обязательного заполнения можно атрибутом required.
